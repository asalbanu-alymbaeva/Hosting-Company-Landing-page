# Hosting Company Landing Page

A modern and responsive landing page designed for a hosting and domain service company.  
This project is fully built using **pure HTML and CSS**, focusing on clean layout, structured sections, and modern UI design.

## 🚀 Live Preview
You can view the live demo here:  
👉 https://splendorous-cactus-aacc14.netlify.app/

## 🛠️ Technologies Used
- HTML5 (semantic structure)
- CSS3
  - Flexbox layout
  - Custom backgrounds
  - Gradients
  - Responsive styling

## ✨ Features
- Responsive landing page design
- Clean and semantic HTML structure
- Custom CSS styling without frameworks
- Flexbox-based layouts
- Reusable and organized CSS code
- Modern UI suitable for hosting and domain services
